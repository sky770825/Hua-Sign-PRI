## ✅ API 路由

### 會員相關
- [x] `GET /api/members` - 獲取所有會員
- [x] `POST /api/members/create` - 新增會員
- [x] `PUT /api/members/[id]` - 更新會員
- [x] `DELETE /api/members/[id]` - 刪除會員

### 會議相關
- [x] `GET /api/meetings` - 獲取所有會議
- [x] `POST /api/meetings` - 新增會議
- [x] `PUT /api/meetings/[id]` - 更新會議
- [x] `DELETE /api/meetings/[id]` - 刪除會議

### 簽到相關
- [x] `GET /api/checkins?date=xxx` - 獲取簽到記錄
- [x] `POST /api/checkin` - 簽到操作
- [x] `POST /api/checkin/delete` - 刪除簽到記錄

## 🔍 需要測試的功能

1. **前台登入**
   - 測試正確密碼登入
   - 測試錯誤密碼提示
   - 測試登入後跳轉

2. **前台簽到**
   - 測試會員簽到
   - 測試簽到留言
   - 測試簽到狀態更新

3. **後台登入**
   - 測試密碼 h123 登入
   - 測試未登入時自動跳轉

4. **後台出席管理**
   - 測試日期選擇
   - 測試建立會議
   - 測試手動簽到
   - 測試刪除簽到記錄
   - 測試統計數據更新

5. **後台會員管理**
   - 測試新增會員
   - 測試編輯會員
   - 測試刪除會員

6. **後台會議管理**
   - 測試新增會議
   - 測試編輯會議
   - 測試刪除會議（應同時刪除簽到記錄）

7. **統計報表**
   - 測試數據計算正確性

## 📝 注意事項

1. 首次執行時會自動建立資料庫和載入會員資料
2. 資料庫檔案位置：`data/checkin.db`
3. 前台密碼：`admin` 或 `123456`
4. 後台密碼：`h123`
5. 所有刪除操作都有確認提示
6. 編輯和新增功能使用彈出式視窗
